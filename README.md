# Wallpaper Mate

一款基于 Tauri 构建的 macOS 壁纸管理器。自动获取、生成并设置精美的壁纸，提供接近原生的使用体验。

## 功能特性

- 🖼️ **壁纸获取** - 从多个来源下载壁纸
- 🎨 **壁纸生成** - AI 驱动的壁纸生成（即将推出）
- ⏰ **自动更新** - 定时自动更换壁纸
- 🎯 **菜单栏集成** - 原生 macOS 菜单栏应用体验
- 💾 **本地收藏** - 保存和管理你喜欢的壁纸

## 技术栈

- **前端**: React/Vue/Svelte (待定)
- **后端**: Rust + Tauri 2.0
- **系统集成**: `wallpaper` crate 用于跨平台壁纸管理
